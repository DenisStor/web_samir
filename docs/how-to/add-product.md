# Как добавить товар

## Через админку

### 1. Откройте админку

http://localhost:8000/admin.html

### 2. Перейдите в "Магазин → Товары"

### 3. Нажмите "Добавить"

### 4. Заполните форму

| Поле | Описание | Обязательное |
|------|----------|--------------|
| Название | Название товара | Да |
| Категория | Выберите из списка | Да |
| Цена | Число в рублях | Да |
| Старая цена | Для отображения скидки | Нет |
| Описание | Полное описание | Нет |
| Изображения | До 5 фото | Да (минимум 1) |
| В наличии | Чекбокс | Да |

### 5. Загрузите изображения

Первое изображение будет главным в карточке товара.

### 6. Сохраните

Товар появится в магазине: http://localhost:8000/shop.html

---

## Создание категории товаров

### 1. Перейдите в "Магазин → Категории"

### 2. Нажмите "Добавить"

### 3. Заполните

| Поле | Описание |
|------|----------|
| Название | Название категории |
| Slug | URL-идентификатор (генерируется автоматически) |

### 4. Сохраните

---

## Структура данных

Категории и товары хранятся в SQLite (`data/saysbarbers.db`), таблицы `shop_categories` и `products`.

---

## Скидки

Для отображения скидки заполните поле "Старая цена":

```json
{
  "price": 1200,
  "oldPrice": 1500
}
```

На сайте отобразится:
- Зачёркнутая старая цена: ~~1 500 ₽~~
- Актуальная цена: **1 200 ₽**
- Бейдж: -20%

---

## Множественные изображения

Товар может иметь до 5 изображений:

```json
"images": [
  "/uploads/product-main.jpg",
  "/uploads/product-side.jpg",
  "/uploads/product-back.jpg"
]
```

Первое изображение — главное (показывается в каталоге).

---

## Товар "Нет в наличии"

Снимите галочку "В наличии" или установите:

```json
"inStock": false
```

Товар будет отображаться с пометкой "Нет в наличии".

---

## Ручное редактирование

Данные хранятся в SQLite. Рекомендуется использовать админку (`/admin.html`).

При необходимости прямого доступа:

```bash
ssh root@80.90.187.187 "sqlite3 /var/www/web_samir-data/data/saysbarbers.db '.tables'"
```

## Частые вопросы

### Как удалить товар?

В админке нажмите иконку корзины на карточке товара.

### Товар не появляется в магазине

1. Проверьте, что товар привязан к категории
2. Проверьте, что категория существует
3. Обновите страницу магазина

### Как изменить порядок товаров?

Перетащите карточку товара мышкой в админке.
