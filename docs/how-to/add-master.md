# Как добавить мастера

## Через админку

### 1. Откройте админку

http://localhost:8000/admin.html

### 2. Перейдите в "Мастера"

Нажмите "Мастера" в боковом меню.

### 3. Нажмите "Добавить"

### 4. Заполните форму

| Поле | Описание | Обязательное |
|------|----------|--------------|
| Имя | Полное имя | Да |
| Роль | Должность (Барбер, Топ-барбер) | Да |
| Badge | Уровень: Green, Pink, Dark Blue | Да |
| Специализация | Что умеет | Нет |
| Фото | JPG/PNG до 5MB | Нет |

### 5. Загрузите фото

Нажмите на область загрузки или перетащите файл.

### 6. Сохраните

Мастер появится на сайте в секции "Команда".

---

## Уровни мастеров (Badge)

| Badge | Описание | Цвет |
|-------|----------|------|
| Green | Начинающий мастер | Зелёный |
| Pink | Опытный мастер | Розовый |
| Dark Blue | Топ-мастер | Синий |

---

## Структура данных

Мастера хранятся в `data/masters.json`:

```json
{
  "masters": [
    {
      "id": "master_1705312847123_abc",
      "name": "Иван Петров",
      "role": "Топ-барбер",
      "badge": "blue",
      "specialization": "Классические стрижки, оформление бороды",
      "photo": "/uploads/ivan.jpg",
      "initial": "И",
      "order": 0
    }
  ]
}
```

---

## Изменение порядка

Перетащите карточку мастера мышкой. Порядок сохраняется автоматически.

---

## Ручное редактирование

### На сервере

```bash
ssh root@80.90.187.187 "nano /var/www/web_samir-data/data/masters.json"
```

### Формат записи

```json
{
  "id": "master_<timestamp>_<random>",
  "name": "Имя Фамилия",
  "role": "Барбер",
  "badge": "green",
  "specialization": "Описание специализации",
  "photo": "/uploads/photo.jpg",
  "initial": "И",
  "order": 0
}
```

**Поля:**
- `id` — уникальный идентификатор
- `badge` — `green`, `pink` или `blue`
- `photo` — путь к фото (опционально)
- `initial` — первая буква имени (показывается, если нет фото)
- `order` — порядок отображения (0, 1, 2...)

---

## Загрузка фото

### Требования

- Формат: JPG, PNG, GIF, WebP
- Размер: до 5MB
- Рекомендуемые пропорции: 1:1 (квадрат)

### Путь к фото

После загрузки фото сохраняется в `uploads/` и путь записывается в JSON:

```json
"photo": "/uploads/master_1705312847123_abc.jpg"
```

---

## Удаление мастера

1. В админке найдите карточку мастера
2. Нажмите иконку корзины
3. Подтвердите удаление

Фото удаляется автоматически.

---

## Частые вопросы

### Фото не загружается

1. Проверьте размер (< 5MB)
2. Проверьте формат (JPG, PNG, GIF, WebP)
3. Проверьте права на папку `uploads/`

### Мастер не появляется на сайте

1. Проверьте, что данные сохранились (нет ошибок в консоли)
2. Обновите страницу (Cmd+Shift+R)
3. Проверьте API: http://localhost:8000/api/masters
