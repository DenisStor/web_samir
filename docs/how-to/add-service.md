# Как добавить услугу

## Через админку (рекомендуется)

### 1. Откройте админку

http://localhost:8000/admin.html

### 2. Войдите

Введите пароль из `config.json` → `admin.password`

### 3. Перейдите в "Услуги"

Нажмите "Услуги" в боковом меню.

### 4. Нажмите "Добавить"

Кнопка в правом верхнем углу.

### 5. Заполните форму

| Поле | Описание | Обязательное |
|------|----------|--------------|
| Категория | Выберите существующую или создайте | Да |
| Название | Название услуги | Да |
| Цена | Число в рублях | Да |
| Описание | Краткое описание | Нет |

### 6. Сохраните

Нажмите "Сохранить". Услуга появится на сайте автоматически.

---

## Структура данных

Услуги хранятся в `data/services.json`:

```json
{
  "categories": [
    {
      "id": "category_1705312847123_abc",
      "name": "Стрижки",
      "order": 0,
      "services": [
        {
          "id": "service_1705312847124_xyz",
          "name": "Мужская стрижка",
          "price": 1500,
          "description": "Стрижка + укладка"
        }
      ]
    }
  ]
}
```

---

## Создание категории

### 1. В админке нажмите "Добавить категорию"

### 2. Введите название

Например: "Стрижки", "Борода", "Комплексы"

### 3. Сохраните

Категория появится в списке. Теперь можно добавлять в неё услуги.

---

## Редактирование порядка

Перетащите услугу или категорию мышкой для изменения порядка.

Порядок сохраняется автоматически.

---

## Ручное редактирование (для продвинутых)

### Локально

```bash
nano data/services.json
```

### На сервере

```bash
ssh root@80.90.187.187 "nano /var/www/web_samir-data/data/services.json"
```

### Формат записи

```json
{
  "id": "service_<timestamp>_<random>",
  "name": "Название услуги",
  "price": 1500,
  "description": "Описание (опционально)"
}
```

**Важно:** ID должен быть уникальным. Используйте формат `service_<timestamp>_<random>`.

---

## Проверка

После добавления:

1. Откройте http://localhost:8000
2. Прокрутите до секции "Услуги"
3. Убедитесь, что услуга отображается

---

## Частые вопросы

### Как скрыть услугу?

Удалите её через админку. Альтернатива — добавьте поле `"hidden": true` в JSON.

### Как изменить порядок категорий?

Перетащите категорию мышкой в админке.

### Услуга не появляется на сайте

1. Проверьте, что сервер запущен (`python3 run.py`)
2. Очистите кэш браузера
3. Проверьте DevTools → Network на ошибки
